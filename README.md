# Aethel — React (Vite + TypeScript)

Luxury skincare landing experience: responsive layout, side menu, section animations, newsletter form validation, React Router (`/` and `/cart`).

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Assets

Place images in `public/assets/images/`:

- `banner.jfif`
- `facecare.jfif`
- `bodycare.jfif`
- `haircare.jfif`

## Stack

- React 19
- TypeScript
- Vite
- React Router
